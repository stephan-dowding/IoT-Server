# IoT-Server

## Installation

1. git clone https://github.com/stephan-dowding/IoT-Server.git 
2. cd IoT-Server 
3. npm install

## Usage - Run Server

1. npm start 
2. Launch browser and go to localhost:3000

## Usage - Run Client

1. Launch browser and go to localhost:3000/countdown/5 to start a countdown timer of 5 minutes
